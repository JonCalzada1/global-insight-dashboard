# Global Insight Dashboard

Global Insight Dashboard is a Python Streamlit app that lets users search for a country and view:

- country information
- live weather data
- map location
- recent news
- favorites and search history

## Features

- Search for a country
- View country details and flag
- See weather metrics
- Explore map-based location context
- Read related news articles
- Save favorites
- Track search history

## Tech Stack

- Python
- Streamlit
- REST Countries API
- Open-Meteo API
- NewsAPI
